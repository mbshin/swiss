# 🏌️ 알프스 7박 8일 골프 트립 (오스트리아 · 프랑스 · 스위스)

**2026년 7월 4일 (토) ~ 7월 11일 (토)** · 40대 남성 2인 · **4라운드 (1라운드/일)**

취리히 도착 → Bregenz(오스트리아) 1박 → 제펠트(티롤) 2박 → 안시(프랑스) 2박 → 제네바(스위스) 2박 → 제네바 출국

> 트레킹/체르마트·크랑몬타나·인스부르크 라운드 폐기, 골프 + 미식 + 호숫가 도시 중심 재구성. 7/7은 라운드 없는 단순 이동일로 부담 분산. 7/10 AI For Good 컨퍼런스 마지막날 참석은 유지. 스위스 라운드 0회 + 스위스 박은 제네바 2박만.
