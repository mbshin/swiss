- **Lufthansa (LH 718)**: [lufthansa.com](https://www.lufthansa.com) 예약 관리 → "myOffer" 입찰 업그레이드 (출발 약 72시간 전~30일 전 이메일 안내). 편도 기준 €300~800 수준 입찰 가능
- **공항 카운터**: 체크인 시 빈 좌석이 있으면 현장 유상 업그레이드 가능
